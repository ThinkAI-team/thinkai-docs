  --------------------------------------------------------------------------
  TC-GOOGLE-01: Đăng nhập Google thành công (User mới)
  --------------------------------------------------------------------------
  Mục đích: Kiểm tra đăng nhập bằng Google với user chưa tồn tại
  Mức độ: CRITICAL
  
  [Bruno Setup]
  - Method: POST
  - URL: http://localhost:8081/auth/google
  - Tab Headers: Content-Type = application/json
  - Tab Body → JSON:
     
     {
       "idToken": "eyJhbGciOiJSUzI1NiIsInR5cCI6IkpXVCJ9..."
     }
     
  [Giải thích idToken]
  - idToken là Google ID Token được trả về từ Google Sign-In frontend
  - Token này chứa: sub (Google ID), email, name, picture
  
  [Expected Response]
  - Status: 200 OK
  - Body:
     {
       "token": "eyJhbGciOiJIUzI1NiJ9...",
       "email": "user@gmail.com",
       "fullName": "User Name",
       "role": "STUDENT",
       "hasPassword": false,
       "isGoogleUser": true,
       "avatarUrl": "https://lh3.googleusercontent.com/..."
     }
  
  [Kiểm tra thêm]
  ✓ Token không rỗng
  ✓ isGoogleUser = true
  ✓ hasPassword = false (user chưa có password)
  ✓ role mặc định là "STUDENT"
  ✓ avatarUrl từ Google

  --------------------------------------------------------------------------
  TC-GOOGLE-02: Đăng nhập Google với email đã tồn tại
  --------------------------------------------------------------------------
  Mục đích: Kiểm tra liên kết Google vào tài khoản hiện có
  Mức độ: CRITICAL
  Tiền điều kiện: Đã có user đăng ký bằng email trong hệ thống
  
  [Bruno Setup]
  - Method: POST
  - URL: http://localhost:8081/auth/google
  - Body → JSON:
     
     {
       "idToken": "eyJhbGciOiJSUzI1NiIsInR5cCI6IkpXVCJ9..."
     }
     
  [Giả định]
  - Email trong idToken = "minh@thinkai.vn" (đã đăng ký trước đó)
  
  [Expected Response]
  - Status: 200 OK
  - Body:
     {
       "token": "eyJhbGciOiJIUzI1NiJ9...",
       "email": "minh@thinkai.vn",
       "fullName": "Bình Minh",
       "role": "STUDENT",
       "hasPassword": true,
       "isGoogleUser": true,
       "avatarUrl": "https://lh3.googleusercontent.com/..."
     }
  
  [Kiểm tra thêm]
  ✓ isGoogleUser = true
  ✓ hasPassword = true (vì đã đăng ký bằng email trước)
  ✓ google_id đã được liên kết vào user

  --------------------------------------------------------------------------
  TC-GOOGLE-03: Đăng nhập Google với user bị khóa

  --------------------------------------------------------------------------
  TC-GOOGLE-04: Google token không hợp lệ
  --------------------------------------------------------------------------
  Mục đích: Kiểm tra verify token thất bại
  Mức độ: HIGH
  
  [Bruno Setup]
  - Method: POST
  - URL: http://localhost:8081/auth/google
  - Body → JSON:
     
     {
       "idToken": "invalid.token.here"
     }
     
  [Expected Response]
  - Status: 401 Unauthorized
  - Body:
     {
       "status": 401,
       "message": "Google token không hợp lệ hoặc đã hết hạn"
     }

  --------------------------------------------------------------------------
  TC-GOOGLE-05: Google token hết hạn
  --------------------------------------------------------------------------
  Mục đích: Kiểm tra xử lý token hết hạn
  Mức độ: HIGH

     {
       "idToken": "eyJhbGciOiJSUzI1NiIsInR5cCI6IkpXVCJ9.eyJzdWIiOiIxMjM0NTY3ODkwIiwiZXhwIjoxNjAwMDAwMDAwfQ.expired"
     }
     
  [Expected Response]
  - Status: 401 Unauthorized
  - Body chứa: "Google token không hợp lệ hoặc đã hết hạn"

  --------------------------------------------------------------------------
  DB-GOOGLE-01: Kiểm tra user mới sau khi đăng nhập Google
  --------------------------------------------------------------------------
  Tiền điều kiện: Đã chạy TC-GOOGLE-01 thành công
  
  Lệnh:
     
     docker exec -it thinkai-mysql mysql -u root -proot -e "USE thinkai_db; SELECT id, email, full_name, google_id, role, is_active FROM users WHERE google_id IS NOT NULL;"
  
  Kiểm tra:
  ✓ google_id có giá trị (không NULL)
  ✓ email được lấy từ Google
  ✓ full_name được lấy từ Google
  ✓ role = "STUDENT"
  ✓ is_active = 1

  --------------------------------------------------------------------------
  DB-GOOGLE-02: Kiểm tra user đã liên kết Google
  --------------------------------------------------------------------------

  Lệnh:
     
     docker exec -it thinkai-mysql mysql -u root -proot -e "USE thinkai_db; SELECT email, google_id, password_hash FROM users WHERE email = 'minh@thinkai.vn';"
  
  Kiểm tra:
  ✓ google_id có giá trị (đã liên kết)
  ✓ password_hash vẫn tồn tại (user có password)
