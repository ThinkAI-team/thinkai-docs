  --------------------------------------------------------------------------
  TC-AUTHME-01: Lấy thông tin user thành công (Happy Path)
  --------------------------------------------------------------------------
  Mục đích: Kiểm tra lấy thông tin user đang đăng nhập
  Mức độ: CRITICAL
  Tiền điều kiện: Có token hợp lệ từ đăng nhập/register
  
  [Bruno Setup]
  - Method: GET
  - URL: http://localhost:8081/auth/me
  - Tab Headers:
    • Content-Type = application/json
    • Authorization = Bearer <TOKEN>
  
  [Expected Response]
  - Status: 200 OK
  - Body:
     {
       "token": "eyJhbGciOiJIUzI1NiJ9...",
       "email": "authtest@thinkai.vn",
       "fullName": "Test User",
       "role": "STUDENT",
       "hasPassword": true,
       "isGoogleUser": false
     }
  
  [Kiểm tra thêm]
  ✓ token trả về là token mới (có thể refresh)
  ✓ email, fullName, role khớp với user đăng nhập
  ✓ hasPassword = true (đăng ký bằng email)
  ✓ isGoogleUser = false

     # Gọi API
     curl -s http://localhost:8081/auth/me \
       -H "Authorization: Bearer $TOKEN" \
