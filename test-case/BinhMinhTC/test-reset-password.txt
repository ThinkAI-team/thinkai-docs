  --------------------------------------------------------------------------
  DB-RESET-01: Kiểm tra password đã được cập nhật
  --------------------------------------------------------------------------
  Tiền điều kiện: Đã chạy TC-RESET-01 thành công
  
  Lệnh:
     
     docker exec -it thinkai-mysql mysql -u root -proot -e "USE thinkai_db; SELECT id, email, password_hash FROM users WHERE email = 'resetTest@thinkai.vn';"
  
  Kiểm tra:
  ✓ password_hash đã thay đổi (khác password cũ)
  ✓ password_hash bắt đầu bằng "$2a$" (BCrypt)

  --------------------------------------------------------------------------
  DB-RESET-02: Kiểm tra token đã được đánh dấu used
  --------------------------------------------------------------------------
  Tiền điều kiện: Đã chạy TC-RESET-01 thành công
  
  Lệnh:
     
     docker exec -it thinkai-mysql mysql -u root -proot -e "USE thinkai_db; SELECT token, used FROM password_reset_tokens WHERE user_id = (SELECT id FROM users WHERE email = 'resetTest@thinkai.vn') ORDER BY created_at DESC LIMIT 1;"
  
  Kiểm tra:
  ✓ used = 1 (true)
