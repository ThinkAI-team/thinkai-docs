────────────────────────────────────────────────────────────────────────────────
  PREREQUISITES
────────────────────────────────────────────────────────────────────────────────

  1. Docker container thinkai-mysql đang chạy:
     docker ps | grep thinkai-mysql

  2. Spring Boot đang chạy trên port 8080:
     cd thinkai-backend && ./mvnw spring-boot:run

  3. Gmail SMTP đã config trong application.properties:
     spring.mail.username=your-email@gmail.com
     spring.mail.password=your-app-password

  4. Có ít nhất 1 user trong DB (email: minh@thinkai.vn)

================================================================================
                    PART 1: FORGOT PASSWORD (Yêu cầu reset)
                    POST /auth/forgot-password
================================================================================

────────────────────────────────────────────────────────────────────────────────
  TC-FP-01: Yêu cầu reset password thành công (email tồn tại)
────────────────────────────────────────────────────────────────────────────────

  Request:
  curl -s -X POST http://localhost:8080/auth/forgot-password \
    -H "Content-Type: application/json" \
    -d '{
      "email": "minh@thinkai.vn"
    }'

  Expected: 200 OK
  {
    "message": "Nếu email tồn tại, chúng tôi đã gửi link đặt lại mật khẩu."
  }

  ✅ Kiểm tra: Mở Gmail → Tìm email từ ThinkAI → Có link reset

────────────────────────────────────────────────────────────────────────────────
  TC-FP-02: Email không tồn tại (vẫn trả 200 - chống email enumeration)
────────────────────────────────────────────────────────────────────────────────

  Request:
  curl -s -X POST http://localhost:8080/auth/forgot-password \
    -H "Content-Type: application/json" \
    -d '{
      "email": "khongtontai@thinkai.vn"
    }'

  Expected: 200 OK
  {
    "message": "Nếu email tồn tại, chúng tôi đã gửi link đặt lại mật khẩu."
  }

  ⚠️  LƯU Ý: Trả cùng response với TC-FP-01 → không tiết lộ email có tồn tại
  ⚠️  KHÔNG gửi email thật (vì email không tồn tại trong DB)

────────────────────────────────────────────────────────────────────────────────
  TC-FP-03: Email rỗng
────────────────────────────────────────────────────────────────────────────────

  Request:
  curl -s -X POST http://localhost:8080/auth/forgot-password \
    -H "Content-Type: application/json" \
    -d '{
      "email": ""
    }'

  Expected: 400 Bad Request
  {
    "error": "Validation Failed",
    "errors": { "email": "Email không được để trống" },
    "status": 400
  }

────────────────────────────────────────────────────────────────────────────────
  TC-FP-04: Email sai format
────────────────────────────────────────────────────────────────────────────────

  Request:
  curl -s -X POST http://localhost:8080/auth/forgot-password \
    -H "Content-Type: application/json" \
    -d '{
      "email": "khong-phai-email"
    }'

  Expected: 400 Bad Request
  {
    "error": "Validation Failed",
    "errors": { "email": "Email không hợp lệ" },
    "status": 400
  }

────────────────────────────────────────────────────────────────────────────────
  TC-FP-05: Rate limit — gửi quá 3 lần/giờ
────────────────────────────────────────────────────────────────────────────────

  Setup: Gửi TC-FP-01 liên tiếp 3 lần trước, rồi gửi lần thứ 4:

  Request (lần thứ 4):
  curl -s -X POST http://localhost:8080/auth/forgot-password \
    -H "Content-Type: application/json" \
    -d '{
      "email": "minh@thinkai.vn"
    }'

  Expected: 429 Too Many Requests
  {
    "error": "Too Many Requests",
    "message": "Bạn đã yêu cầu quá nhiều lần. Vui lòng thử lại sau 1 giờ.",
    "status": 429
  }

────────────────────────────────────────────────────────────────────────────────
  TC-FP-06: Body rỗng
────────────────────────────────────────────────────────────────────────────────

  Request:
  curl -s -X POST http://localhost:8080/auth/forgot-password \
    -H "Content-Type: application/json" \
    -d '{}'

  Expected: 400 Bad Request
  {
    "error": "Validation Failed",
    "errors": { "email": "Email không được để trống" },
    "status": 400
  }

================================================================================
                    PART 2: RESET PASSWORD (Đặt lại mật khẩu)
                    POST /auth/reset-password
================================================================================

  ⚠️  Để test Part 2, cần lấy token từ email (TC-FP-01) hoặc query DB:
     SELECT token FROM password_reset_tokens ORDER BY id DESC LIMIT 1;

────────────────────────────────────────────────────────────────────────────────
  TC-RP-01: Reset password thành công
────────────────────────────────────────────────────────────────────────────────

  Setup: Lấy token từ TC-FP-01

  Request:
  curl -s -X POST http://localhost:8080/auth/reset-password \
    -H "Content-Type: application/json" \
    -d '{
      "token": "<TOKEN_TỪ_EMAIL>",
      "newPassword": "NewPass123",
      "confirmPassword": "NewPass123"
    }'

  Expected: 200 OK
  {
    "message": "Đặt lại mật khẩu thành công. Vui lòng đăng nhập."
  }

  ✅ Xác minh: Đăng nhập bằng mật khẩu mới "NewPass123"
  curl -s -X POST http://localhost:8080/auth/login \
    -H "Content-Type: application/json" \
    -d '{
      "email": "minh@thinkai.vn",
      "password": "NewPass123"
    }'

────────────────────────────────────────────────────────────────────────────────
  TC-RP-02: Token đã được sử dụng (dùng lại token từ TC-RP-01)
────────────────────────────────────────────────────────────────────────────────

  Request (dùng lại cùng token):
  curl -s -X POST http://localhost:8080/auth/reset-password \
    -H "Content-Type: application/json" \
    -d '{
      "token": "<TOKEN_ĐÃ_DÙNG>",
      "newPassword": "AnotherPass123",
      "confirmPassword": "AnotherPass123"
    }'

  Expected: 400 Bad Request
  {
    "error": "Bad Request",
    "message": "Link đặt lại mật khẩu đã được sử dụng",
    "status": 400
  }

────────────────────────────────────────────────────────────────────────────────
  TC-RP-03: Token không hợp lệ (random UUID)
────────────────────────────────────────────────────────────────────────────────

  Request:
  curl -s -X POST http://localhost:8080/auth/reset-password \
    -H "Content-Type: application/json" \
    -d '{
      "token": "fake-token-12345-random",
      "newPassword": "NewPass123",
      "confirmPassword": "NewPass123"
    }'

  Expected: 400 Bad Request
  {
    "error": "Bad Request",
    "message": "Link đặt lại mật khẩu không hợp lệ",
    "status": 400
  }

────────────────────────────────────────────────────────────────────────────────
  TC-RP-04: Token hết hạn (cần chờ 30 phút hoặc update DB)
────────────────────────────────────────────────────────────────────────────────

  Setup: Tạo token rồi cập nhật DB cho hết hạn:
    UPDATE password_reset_tokens
    SET expires_at = NOW() - INTERVAL 1 HOUR
    WHERE token = '<TOKEN>';

  Request:
  curl -s -X POST http://localhost:8080/auth/reset-password \
    -H "Content-Type: application/json" \
    -d '{
      "token": "<TOKEN_HẾT_HẠN>",
      "newPassword": "NewPass123",
      "confirmPassword": "NewPass123"
    }'

  Expected: 400 Bad Request
  {
    "error": "Bad Request",
    "message": "Link đặt lại mật khẩu đã hết hạn. Vui lòng yêu cầu lại.",
    "status": 400
  }

────────────────────────────────────────────────────────────────────────────────
  TC-RP-05: Mật khẩu xác nhận không khớp
────────────────────────────────────────────────────────────────────────────────

  Request:
  curl -s -X POST http://localhost:8080/auth/reset-password \
    -H "Content-Type: application/json" \
    -d '{
      "token": "<TOKEN_HỢP_LỆ>",
      "newPassword": "NewPass123",
      "confirmPassword": "KhacPassword456"
    }'

  Expected: 400 Bad Request
  {
    "error": "Bad Request",
    "message": "Mật khẩu xác nhận không khớp",
    "status": 400
  }

────────────────────────────────────────────────────────────────────────────────
  TC-RP-06: Mật khẩu mới quá ngắn (< 8 ký tự)
────────────────────────────────────────────────────────────────────────────────

  Request:
  curl -s -X POST http://localhost:8080/auth/reset-password \
    -H "Content-Type: application/json" \
    -d '{
      "token": "<TOKEN_HỢP_LỆ>",
      "newPassword": "abc",
      "confirmPassword": "abc"
    }'

  Expected: 400 Bad Request
  {
    "error": "Validation Failed",
    "errors": { "newPassword": "Mật khẩu mới phải có ít nhất 8 ký tự" },
    "status": 400
  }

────────────────────────────────────────────────────────────────────────────────
  TC-RP-07: Token rỗng
────────────────────────────────────────────────────────────────────────────────

  Request:
  curl -s -X POST http://localhost:8080/auth/reset-password \
    -H "Content-Type: application/json" \
    -d '{
      "token": "",
      "newPassword": "NewPass123",
      "confirmPassword": "NewPass123"
    }'

  Expected: 400 Bad Request
  {
    "error": "Validation Failed",
    "errors": { "token": "Token không được để trống" },
    "status": 400
  }

────────────────────────────────────────────────────────────────────────────────
  TC-RP-08: Body rỗng
────────────────────────────────────────────────────────────────────────────────

  Request:
  curl -s -X POST http://localhost:8080/auth/reset-password \
    -H "Content-Type: application/json" \
    -d '{}'

  Expected: 400 Bad Request
  {
    "error": "Validation Failed",
    "errors": {
      "token": "Token không được để trống",
      "newPassword": "Mật khẩu mới không được để trống",
      "confirmPassword": "Xác nhận mật khẩu không được để trống"
    },

                            GHI CHÚ
================================================================================

  • TC-FP-01 và TC-FP-02 trả CÙNG response → chống email enumeration
  • Token là UUID, hết hạn 30 phút, dùng 1 lần
  • Rate limit: max 3 request/email/giờ
  • Email gửi qua Gmail SMTP → kiểm tra cả thư mục Spam
  • Link reset: http://localhost:3000/reset-password?token=<UUID>
  • Sau khi reset thành công → đăng nhập bằng mật khẩu mới
