### 5.2. Feedback & Microinteractions

| Action | Feedback |
|--------|----------|
| Button click | Ripple effect (optional) |
| Hover | Subtle scale/shadow change |
| Form submit success | Toast + redirect |
| File upload | Progress bar |
| Toggle | Smooth transition |

### 5.3. Accessibility Guidelines

| Requirement | Implementation |
|-------------|----------------|
| **Keyboard Navigation** | All interactive elements focusable |
| **Focus Visible** | Clear focus ring (`outline: 2px solid primary`) |
| **Color Contrast** | Min 4.5:1 for text |
| **Alt Text** | All images have descriptive alt |
| **ARIA Labels** | For icons and non-text elements |
| **Error Identification** | Not rely on color alone |

### 5.4. Responsive Breakpoints

| Breakpoint | Width | Layout Changes |
|------------|-------|----------------|
| `sm` | 640px | Stack cards, hide sidebar |
| `md` | 768px | 2-column grid, collapsible sidebar |
| `lg` | 1024px | Full layout, fixed sidebar |
| `xl` | 1280px | Max container width |

### 5.5. Animation Guidelines

```css
/* Duration */
--duration-fast: 150ms;    /* Hover, focus */
--duration-normal: 200ms;  /* Transitions */
--duration-slow: 300ms;    /* Modal, Page */

/* Easing */
--ease-in-out: cubic-bezier(0.4, 0, 0.2, 1);
--ease-out: cubic-bezier(0, 0, 0.2, 1);
--ease-in: cubic-bezier(0.4, 0, 1, 1);

/* Principles */
/* 1. Keep animations < 300ms */
/* 2. Use ease-out for entering, ease-in for leaving */
/* 3. Reduce motion for users who prefer it */
```

```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    transition-duration: 0.01ms !important;
  }
}
```

## 7. Dark Mode

### 7.1. Color Mapping

| Light Token | Dark Token |
|-------------|------------|
| `--neutral-0` (bg) | `--dark-bg-primary` |
| `--neutral-50` (card) | `--dark-bg-secondary` |
| `--neutral-700` (text) | `--dark-text-primary` |
| `--neutral-500` (secondary) | `--dark-text-secondary` |
| `--neutral-100` (border) | `--dark-border` |

### 7.2. Implementation

```css
/* System preference */
@media (prefers-color-scheme: dark) {
  :root {
    color-scheme: dark;
    /* Apply dark tokens */
  }
}

/* Manual toggle */
[data-theme="dark"] {
  /* Apply dark tokens */
}
```
