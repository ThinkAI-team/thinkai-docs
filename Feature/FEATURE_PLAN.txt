1. BÌNH MINH — Module: AUTHENTICATION & USER PROFILE
--------------------------------------------------------------------------------
   Làm ĐẦU TIÊN vì các module khác phụ thuộc vào Auth!

   [1] Đăng ký tài khoản (Email/Password)
       - API: POST /auth/register
       - Màn hình: Register Page
       - Độ khó: ⭐⭐

   [2] Đăng nhập (Email/Password)
       - API: POST /auth/login
       - Màn hình: Login Page
       - Độ khó: ⭐⭐

   [3] Đăng nhập Google OAuth
       - API: POST /auth/google
       - Màn hình: Login Page (nút Google)
       - Độ khó: ⭐⭐⭐

   [4] Quản lý Profile (Xem/Sửa thông tin cá nhân)
       - API: GET/PUT /users/me
       - Màn hình: Profile Page
       - Độ khó: ⭐⭐

   [5] Đổi mật khẩu
       - API: PUT /users/me/password
       - Màn hình: Profile Page
       - Độ khó: ⭐

   >>> Output: Auth Context, JWT handling, Protected routes
